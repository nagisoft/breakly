## Breakly

Breakly is a plateform that brings you bite sized activities to refresh yourself during your lunch or studies break.

## Features

Features Done :

- [x] HomePage
- [x] Deep Breathing App
- [x] Card Memory Game
- [x] Word Of The Day
- [x] Picture Gallery
- [x] Quotes
